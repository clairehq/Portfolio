Project Description

This project is for a web design and development course. It is about to design an e-commerce company from scratch. The goal of this project is to be able to design a decent website with a clear branding idea.

Define the Brand

Yumm is an e-commerce company that focuses on Asian snacks and drinks. Most of the snacks and drinks are from China, Japan, and South Korea. It provides a shopping platform for customers living outside of Asia to buy Asian snacks and drinks at any time. The goal of the company is to attract customers to buy Asian snacks and drinks in an assured way with quality guaranteed and fast delivery guaranteed.

build the website

After we decided the overall style of the website, I started to sketch out the structure of the website. I created a sitemap to see how different pages are linked and sketched the layouts for every page. In order to make the style and resolution of all images consistent, I added the same background and resized them in Sketch. During the coding process, the hardest part of this project for me is to make the website responsive smoothly. I need to test the website several times in different screen sizes and make sure that it breaks well in particular screen size.
